Here is list of my current open-source projects:

Oscillograph

Simple oscilloscope created using Arduino UNO and Qt library. The Arduino simply measures the voltage on analog inputs and sends them via serial port to local PC.

Qt based application reads the serial data and displays the measured value in same way as the oscilloscope does.

BluetoothButton

Use the button(s) connected to Arduino to execute various actions on your mobile phone

This tool was first created for disabled persons to be able to run the phone actions using only the button
